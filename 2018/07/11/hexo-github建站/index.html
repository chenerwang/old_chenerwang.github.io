hexo n “我的博文” #新建文章
hexo generate 也可简写成 hexo g #生成
hexo server 也可简写成 hexo s #启动服务预览（监视文件变动并自动更新，无需重启服务器）
hexo deploy 也可简写成 hexo d #部署
hexo server -s #静态模式
hexo server -p 5000 #修改端口
hexo server -i 192.168.1.1 #自定义IP
hexo clean #清楚缓存
