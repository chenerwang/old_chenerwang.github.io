解释器

  1. Linux设置环境变量

  2. 使Python脚本像shell脚本一样可直接执行

    1. 顶部添加#! /usr/bin/env python3
    2. 修改脚本权限 chmod +x hello.py
    3. 执行：./hello.py

  3. 解释器种类

    CPython、IPython、Jython、PyPy 等

注释

  1. 单行注释

    #开头

  2. 多行注释

    ''' 或 """ 将注释括起来

  3. 输出函数的注释

    函数名.doc

7种运算符

  1. 算术运算符

    +，-，*，/，**，//（取整除），%

  2. 比较运算符

    ==，!=，>，<，>=，<=

  3. 赋值运算符

    = += -= *= /= **= //= %=

  4. 位运算符

    & | ^ ~ << >>

  1. 逻辑运算符

    and or not

    具有短路效果，返回最后一个运算到的值

  2. 成员运算符

    in, not in: 判断是否在序列中

  3. 身份运算符

    • is, is not
    • 判断两个对象的存储单元，即id()对象内存地址
    • is判断对象是否为同一个，==用于判断引用变量的值是否相等

  4. 优先级

    | 运算符 | 描述 |
    | - | :-: | -: |
    | ** | 指数 (最高优先级) |
    | ~ + - | 按位翻转, 一元加号和减号 (最后两个的方法名为 +@ 和 -@) |
    | * / % // | 乘，除，取模和取整除 |
    | + - | 加法减法 |
    | >> << | 右移，左移运算符 |
    | & | 位 ‘AND’ |
    | ^ | | 位运算符 |
    | <= < > >= | 比较运算符 |
    | <> == != | 等于运算符 |
    | = %= /= //= -= += *= **= | 赋值运算符 |
    | is is not | 身份运算符 |
    | in not in | 成员运算符 |
    | and or not | 逻辑运算符 |
